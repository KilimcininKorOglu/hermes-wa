# Run Worker (Development)
# From project root (RECOMMENDED)
go run ./cmd/worker

# Or from the cmd/worker folder
go run .

# Run Worker (Production with PM2)
# 1. Build the binary first
go build -o hermeswa-worker.exe ./cmd/worker  # Windows
go build -o hermeswa-worker ./cmd/worker      # Ubuntu

# 2. Run with PM2
pm2 start cmd/worker/ecosystem.config.js


Important Note: Make sure when you build, the output binaries are placed in the project root folder to match the ecosystem.config.js configuration.

Example build commands:

# Build API
go build -o hermeswa main.go
# Build Worker (Run this from root)
go build -o worker cmd/worker/*.go

After building, simply run:
pm2 start ecosystem.config.js
